Detailed Description

Simulation is the focus of the high-level C++ API of ReaDDy. This is where the system is set up and run for a certain number of time steps. Things like temperature, boxsize, reactions and potentials belong to the context and are given to the kernel when run() is called.

Constructs a Lennard-Jones-type potential between two particle types A and B (where possibly A = B) of the form

\[ V_{\mbox{LJ}}(r) = k(\epsilon , n, m) \left[ \left(\frac{\sigma}{r}\right)^m - \left(\frac{\sigma}{r}\right)^n \right], \]

where n,m are exponent 1 and 2, respectively, with m > n. If shift == true, it will be defined as

\[ V_{\mbox{LJ, shifted}}(r) = V_{\mbox{LJ}}(r) - V_{\mbox{LJ}}(r_{\mbox{cutoff}}) \]

for r <= cutoffDistance, which makes a difference in energy, but not in force.

Parameters
particleType1particle type A
particleType2particle type B
mfirst exponent
nsecond exponent
cutoffDistancethe cutoff distance
shiftif it should be shifted or not
epsilonthe well depth
sigmathe distance at which the inter-particle potential is zero
Returns
a uuid with which the potential can be removed again

Constructs a potential that describes screened electrostatics with a hard-core repulsion between two particle types A and B (where possibly A = B) of the form

\[ V(r) = C \frac{\exp(-\kappa r)}{r} + D\left(\frac{\sigma}{r}\right)^n, \]

where the first term is the electrostatic interaction, the constant C has the dimensions of an energy times distance. Its value can be positive or negative and depends on the valencies of the particles (see Debye-Hueckel theory). \(\kappa\) is the inverse screening depth. The second term is a hard-core repulsion, that ensures that the potential does not diverge to negative infinity.

Parameters
particleType1particle type A
particleType2particle type B
electrostaticStrengthC
inverseScreeningDepth\(\kappa\)
repulsionStrengthD
repulsionDistance\(\sigma\)
exponentn
cutoffthe distance from which no energies and forces are calculated further
Returns
a uuid with which the potential can be removed again
