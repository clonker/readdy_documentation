Detailed Description

template<typename RESULT, typename... PARENT_OBS>
class readdy::model::observables::Combiner< RESULT, PARENT_OBS >

Combiner observable, that unifies a collection of observables and uses their results to compute something. One should take care, that the parent observables strides fit with the stride of this observable or take into account, that some values might be outdated.

Template Parameters
RESULTthe result type, see readdy::model::observables::Observable.
PARENT_OBSthe parent observables which will be evaluated before this observable gets evaluated
